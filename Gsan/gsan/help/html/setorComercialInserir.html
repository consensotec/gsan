Inserir Setor Comercial

Esta opção do sistema permite a inserção de um setor comercial na base de dados.

Preenchimento dos campos

  • Localidade(*): Informe o código de uma localidade existente e tecle "Enter", ou clique no botão "Pesquisar", que fica ao lado do campo. Neste caso será apresentada uma tela de "popup", onde será possível efetuar a pesquisa de localidades no cadastro. Após a informação do código de uma localidade existente, ou da seleção de uma localidade na tela de pesquisa, o sistema apresentará o nome da localidade no campo correspondente.

    • No caso de se informar uma localidade inexistente, o sistema apresentará a mensagem: "Localidade Inexistente"

  • Código do Setor Comercial(*): Informe o código de setor comercial inexistente na localidade informada. Você pode, também, consultar os setores comerciais já existentes na localidade, clicando no botão "Pesquisar"  que fica ao lado do campo.  Será apresentada a tela "Pesquisar Setor Comercial". Após a pesquisa dos setores comerciais, caso você clique sobre o link existente no Código do Setor Comercial, o sistema irá transferir o controle para a opção "Atualizar Setor Comercial"

  • Nome do Setor Comercial(*): Informe o nome do Setor Comercial

  • Município(*): Informe o código de um município existente e tecle "Enter", ou clique no botão "Pesquisar" , que fica ao lado do campo. Neste caso será apresentada a tela "Pesquisar Município", onde será possível efetuar a pesquisa de municípios no cadastro. Após a informação do código de um município existente, ou da seleção de um município na tela de pesquisa, o sistema apresentará o nome do município no campo correspondente. 
    • No caso de se informar um código de município inexistente, o sistema apresentará a mensagem: "Município Inexistente".

Observação: Os campos obrigatórios estão marcados um um asterisco vermelho (*)

Funcionalidade dos botões:

Botão

Descrição

Utilize este botão para ativar as funcionalidades de pesquisa de uma localidade, ou de um município. Deve ser utilizado quando não se conhece o código da informação que deseja inserir. 

Ao clicar no botão, o sistema apresentará uma outra tela, onde será realizada a pesquisa.

Utilize este botão para limpar as informações existentes num determinado campo da tela. Eventualmente, poderá efetuar a limpeza de campos relacionados, com o objetivo de eliminar inconsistências. Por exemplo, nesta tela, ao limpar o campo "Localidade", o sistema limpará, automaticamente, o campo "Código do Setor Comercial".
Utilize este botão para pesquisar os setores comerciais existentes na base de dados, quando você não souber o código do setor comercial. Ao clicar no botão o sistema irá apresentar a tela "Pesquisar Setor Comercial". Após a realização da pesquisa o sistema irá apresentar a relação dos setores comerciais selecionados, classificadas pelo código do setor comercial. Caso você clique no link existente no código do setor comercial, o sistema irá transerir o controle para a funcionalidade "Atualizar Setor Comercial".
Utilize este botão para solicitar ao sistema a inserção do setor comercial na base de dados. Neste caso, é necessário que todos os campos estejam preenchidos corretamente. Caso exista alguma inconsistência, o sistema emitirá a mensagem de crítica correspondente. 
Utilize este botão para fazer com que a tela volte ao seu estado inicial de exibição. 
Utilize este botão para fazer com que o sistema encerre, sem salvar, o que está sendo feito, e volte para a tela principal.

Tela de Sucesso:

A tela de sucesso será apresentada após clicar no botão "Inserir", e não houver nenhuma inconsistência no conteúdo dos campos da tela.
O sistema apresentará a  mensagem abaixo, quando a inclusão do setor comercial tiver sido realizada com sucesso. 

"Setor Comercial de código <<código do setor comercial>>, da localidade <<nome da localidade>>, inserido com sucesso."

O sistema apresentará tres opções após a inseção da quadra. Escolha a opção desejada clicando em algum dos "links" existentes na tela de sucesso:

  • Menu Principal  --> Para voltar à tela prinicipal.
  • Inserir outro Setor Comercial --> Para inserir outra setor comercial.
  • Atualizar Setor Comercial Inserido --> Para efetuar a atualização do setor comercial, recentemente, inserido.

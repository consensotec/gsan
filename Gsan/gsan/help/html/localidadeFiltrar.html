Filtrar Localidade

Esta opção do sistema faz parte do conjunto de funcionalidades responsáveis pela estrutura de localização de imóveis.
  • Localidade
  • Setor Comercial
  • quadra
Esta tela permite que você informe os parâmetros para filtrar as localidades que deseja alterar, excluir ou emitir um relatório. 

Preenchimento dos campos

Informar, pelo menos, um dos parâmetros de pesquisa relacionados abaixo:

  • Atualizar:  Você deve utilizar este campo para decidir se deseja que a tela "Manter Localidade" seja apresentada quando o resultado do filtro retornar apenas um registro do banco de dados. Normalmente, ele deve ficar marcado quando se desejar efetuar uma alteração; e desmarcado quando se desejar efetuar uma exclusão, ou emissão de um relatório com o resultado do filtro.
  • Código: Informe o código da localidade que deseja filtrar. 
  • Nome: Informar o nome, ou parte do nome da localidade. Não se preocupe com acentuação e caracteres especiais; nem com maiúsculas e minúsculas; pois o sistema se encarregará de eliminar estas diferenças. Faça a opção pelo mecanismo de pesquisa a ser aplicado no nome da localidade informado:
    • Iniciando pelo texto --> Significa que o sistema deverá efetuar a pesquisa, considerando que o argumento de pesquisa informado corresponde às primeiras letras do nome da localidade.
    • Contendo o texto --> Significa que o sistema deverá efetuar a pesquisa, considerando que o argumento de pesquisa informado pode estar contido em qualquer parte do nome da localidade.
  • Gerência Regional: Selecione, se for o caso, a Gerência Regional, para a qual deseja limitar o filtro das localidades.

  • Indicador de Uso: Selecione a opção de acordo com suas necessidades em relação à situação da localidade na base de dados. Ativo, significa que só serão pesquisadas as localidades que estiverem ativas. Inativo, as localidades inativas. E Todos, serão apresentadas todas as localidades, independentemente da situação. As localidades inativas serão apresentadas na cor vermelha.

Funcionalidade dos botões:

Botão

Descrição

Utilize este botão para solicitar ao sistema a realização do filtro das localidades na base de dados, de acordo com os argumentos de pesquisa informados.

Utilize este botão para limpar todos os campos da tela.

Tela de Sucesso:

A tela de sucesso vai depender da configuração do filtro.
Se o campo atualizar estiver marcado e o filtro retornar apenas um registro, o sistema vai apresentar a tela "Atualizar Localidade".
Caso contrário, o sistema vai apresentar a tela "Manter Localidade " com a relação das localidades selecionadas pelo filtro.

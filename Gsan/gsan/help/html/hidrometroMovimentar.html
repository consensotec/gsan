Movimentar Hidrômetro

Esta opção do sistema permite que seja efetuada a movimentação de hidrômetros para outro local de armazenagem.

Inicialmente, o sistema apresenta a tela "Filtrar Hidrômetro" para que seja efetuado o filtro do(s) hidrômetros(s) para a movimentação do local de armazenagem. 

Após a execução do filtro, o sistema irá apresentar a tela "Movimentar Hidrômetro", com a relação dos hidrômetros pesquisados; ou irá, direto, para a tela "Movimentação de Hidrômetro(s)".

O sistema irá para a tela "Movimentar Hidrômetro", com a relação dos hidrômetros pesquisados, quando o campo (check-box) "Atualizar" estiver desmarcado, ou quando a quantidade de registros selecionados na pesquisa for maior do que 1. 

O sistema irá, direto, para a tela "Movimentação de Hidrômetro(s)", quando houver a negação da condição relatada acima. Ou seja, quando o campo "Atualizar"  estiver marcado e a quantidade de registros selecionados for igual a 1. 

O quadro, abaixo, apresenta as condições possíveis e a tela a ser apresentada.

Tela Movimentar Hidrômetro

O sistema apresenta uma tabela com a relação dos hidrômetros selecionados, de acordo com os argumentos de pesquisa informados na tela Filtrar Hidrômetro.
A relação dos hidrômetros selecionados estará classificada pelo número do hidrômetro, e tem uma coluna na esquerda para você selecionar os hidrômetros, para os quais, deseja efetuar a movimentação de local de armazenagem.
Caso deseje selecionar todos os hidrômetros de uma única vez, clique no  "hyperlink" "Todos" existente no cabeçalho da tabela. Se clicar, novamente, no "hyperlink", o sistema irá desmarcar todos os hidrômetros, e, assim, sucessivamente.
Após selecionar na tabela os hidrômetros que mudarão de local de armazenagem, clique no botão , para ativar a tela "Movimentação de Hidrômetro(s)", onde você irá informar os dados da movimentação e concluir o processo.

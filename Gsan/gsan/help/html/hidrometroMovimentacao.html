Movimentação de Hidrômetro(s)

Esta tela faz parte do processo "Movimentar Hidrômetro", e permite que você informe os dados da movimentação do local de armazenagem de hidrômetro(s) e conclua o processo.

O sistema apresenta o local de armazenagem atual preenchido e protegido contra atualização e os campos para você informar os dados da movimentação do(s) hidrômetro(s).  Os campos de data e hora da movimentação já vêem previamente preenchidos, mas você poderá alterá-los para os valores que desejar.
Informe os dados da movimentação e clique no botão  para solicitar ao sistema a execução da movimentação na base de dados.

Prenchimento dos campos

  • Local de Armazenagem Destino(*): Informe o código do local de armazenagem de destino do(s) hidrômetro(s) e tecle "Enter", ou clique no botão "Pesquisar" , que fica ao lado do campo. Neste caso será apresentada a tela "Pesquisar Local de Armazenagem de Hidrômetro", onde será possível efetuar a pesquisa dos locais de armazenagem. Após a informação do código do local de armazenagem destino, ou da seleção do local de armazenagem destino na tela de pesquisa, o sistema apresentará o nome do local de armazenagem no campo correspondente. 
    • No caso de se informar um código de local de armazenagem inexistente, o sistema apresentará a mensagem: "Local de Armazenagem Inexistente".

  • Data da Movimentação(*): Informe a data da movimentação do(s) hidrômetro(s). O sistema já coloca como padrão a data corrente, entretanto você poderá modificá-la, desde que seja uma data inferior à data corrente e superior, ou igual, à data de aquisição do(s) hidrômetro(s).

    • Você tem duas opções para informar a data: 
      • Digitando-a no campo. Neste caso você não precisa se preocupar com as barras de separação, pois o sistema as coloca automaticamente. A data deve ser informada no formato: DD/MM/AAAA, onde DD é o dia, MM, o mês e AAAA, o ano. O dia e o mês devem ser informados, obrigatoriamente, com dois dígitos e o ano com quatro. 
      • Pesquisando-a num calendário. Neste caso, clique no botão "Calendário" , existente ao lado do campo, que será apresentada uma tela onde você poderá selecionar uma data no calendário. 
  • Hora da Movimentação(*): Informe hora e minutos da movimentação do hidrômetro. Não se preocupe com os dois pontos entre a hora e os minutos, pois o sistema se encarrega de colocá-los automaticamente. A hora deve ser informada no formato: HH:MM, onde HH é a hora e MM, os minutos. A hora e os minutos devem ser informados, obrigatoriamente, com dois dígitos. 
  • Motivo da Movimentação(*): Selecione, na lista apresentada ao lado do campo, o motivo da movimentação do hidrômetro. 
  • Parecer: Informe um parecer explicando os motivos da movimentação. O texto é livre e deve ter, no máximo, 50 caracteres.

Observação 1: Os campos obrigatórios estão marcados um um asterisco vermelho (*)

Funcionalidade dos botões

Botão

Descrição
Ao acionar este botão, o sistema íra abrir uma tela para que você selecione a data a partir de um calendário. Clique aqui para obter a ajuda da tela do calendário.

Utilize este botão para voltar para a tela anterior.
Utilize este botão para fazer com que a tela volte ao seu estado inicial de exibição. 
Utilize este botão para fazer com que o sistema encerre, sem salvar o que está sendo feito, e volte para a tela principal.
Utilize este botão para solicitar ao sistema a movimentação do local de armazenagem do(s) hidrômetro(s). Neste caso, é necessário que todos os campos estejam preenchidos corretamente. Caso exista alguma inconsistência, o sistema emitirá a mensagem de crítica correspondente. 

Tela de Sucesso:

A tela de sucesso será apresentada após você clicar no botão , e não houver nenhuma inconsistência no conteúdo dos campos da tela.
O sistema apresentará a  mensagem abaixo, quando a mopvimentação do local de armazenagem do(s) hidrômetro(s) tiver sido realizada com sucesso. 

“<<quantidade de hidrômetros movimentados>> Hidrômetro(s) movimentado(s) com sucesso.”

O sistema apresentará duas opções após a movimentação do(s) hidrômetro(s). Escolha a opção desejada clicando em algum dos “hyperlinks” existentes na tela de sucesso:

  • Menu Principal  --> Para voltar à tela prinicipal.
  • Movimentar outro(s) Hidrômetro(s) --> Para efetuar a movimentação de outro(s) hidrômetro(s). Será apresentada a tela "Filtrar Hidrômetro".

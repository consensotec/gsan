Consultar Movimentação de Hidrômetro

Lista das Movimentações

Esta opção do sistema faz parte do processo "Consultar Movimentação de Hidrômetro", que se refere às movimentações de local de armazenagem.

  1. Inicialmente, o sistema apresenta a tela "Filtrar Movimentação de Hidrômetros" para você efetuar o filtro do(s) das movimentações de hidrômetros(s) que deseja consultar. 
  2. Após a execução do filtro, o sistema apresenta a tela "Lista das Movimentações de Hidrômetro" com a relação das movimentações de hidrômetros pesquisadas.
  3. Finalmente será apresentada a tela "Consultar Movimentação de Hidrômetro", com os dados da movimentação.
Tela Lista das Movimentações de Hidrômetro

O sistema apresenta uma tabela com a relação das movimentações de hidrômetros, de acordo com os argumentos de pesquisa informados na tela Filtrar Movimentação de Hidrômetro.
A relação das movimentações de hidrômetros estará classificada pela data da movimentação, que, também, terá um "hyperlink" para que você possa selecionar a movimentação que deseja consultar.
Ao clicar no "hyperlink" da data da movimentação, o sistema irá apresentar a consulta com os dados da movimentação.
Caso, para o critério de pesquisa informado, tenham sido selecionadas mais de 10 movimentações de hidrômetros, o sistema irá montar um mecanismo de paginação no final da página.
Clicando sobre o número da página, você poderá navegar para as demais páginas.

Funcionalidade dos botões

Botão

Funcionalidade
Retornar para a tela de filtro. O sistema irá manter os argumentos de pesquisa informados anteriormente.

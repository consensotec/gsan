Inserir Quadra

Esta opção do sistema permite a inserção de uma quadra na base de dados.

Preenchimento dos campos

  • Localidade(*): Informe o código de uma localidade existente e tecle "Enter", ou clique no botão "Pesquisar", que fica ao lado do campo. Neste caso será apresentada uma tela de "popup", onde será possível efetuar a pesquisa de localidades no cadastro. Após a informação do código de uma localidade existente, ou da seleção de uma localidade na tela de pesquisa, o sistema apresentará o nome da localidade no campo correspondente.

    • No caso de se informar uma localidade inexistente, o sistema apresentará a mensagem: "Localidade Inexistente"

  • Setor Comercial(*): Informe o código de um setor comercial existente na localidade informada e tecle "Enter", ou clique no botão "Pesquisar", que fica ao lado do campo. Neste caso será apresentada uma tela de "popup", onde será possível efetuar a pesquisa de setores comerciais no cadastro. Após a informação do código de um setor comercial existente, ou da seleção de um setor comercial na tela de pesquisa, o sistema apresentará o nome do setor comercial no campo correspondente.

    • No caso de se informar um setor comercial inexistente, o sistema apresentará a mensagem: "Setor Comercial Inexistente"

  • Quadra(*): Informe o código de uma quadra inexistente na localidade e setor comercial informados anteriormente. Você pode, também, consultar as quadras já existentes para a localidade e setor comercial informados, clicando no botão "Pesquisar" que fica ao lado do campo. Neste caso, será apresentada uma tela com a relação de quadras. 

    • Regra para geração do código da quadra: ???

  • Perfil da Quadra(*): Selecione um perfil no "combo-box" correspondente.

  • Rede de Água(*): Informe se não existe, existe, ou se a quadra é parcialmente coberta pela rede de água.

  • Rede de Esgoto(*): Informe se não existe, existe, ou se a quadra é parcialmente coberta pela rede de esgoto.

  • Os campos relacionados abaixo só deverão ser informados, caso exista rede de esgoto na quadra:

    • Sistema de Esgoto: Informe o código do sistema de esgoto, ou selecione no "combo-box" ao lado do campo.

    • Bacia: Informe o código do sistema de esgoto, ou selecione no "combo-box" ao lado do campo.

  • Distrito Operacional: Só deve ser informado, caso exista rede de água na quadra. Neste caso, informe o código de um distrito operacional existente e tecle "Enter", ou clique no botão "Pesquisar", que fica ao lado do campo. Neste caso será apresentada uma tela de "popup", onde será possível efetuar a pesquisa dos distritos operacionais existentes no cadastro. Após a informação do código de um distrito operacional, ou da seleção de um distrito operacional na tela de pesquisa, o sistema apresentará o nome do distrito operacional no campo correspondente.

    • No caso de se informar um distrito operacional inexistente, o sistema apresentará a mensagem: Distrito Operacional Inexistente"

  • Setor Censitário: Informe o código de um setor censitário existente e tecle "Enter", ou clique no botão "Pesquisar", que fica ao lado do campo. Neste caso será apresentada uma tela de "popup", onde será possível efetuar a pesquisa de setores censitáriosno cadastro. Após a informação do código de um setor censitário existente, ou da seleção de um setor censitário na tela de pesquisa, o sistema apresentará o nome do setor censitário no campo correspondente.

    • No caso de se informar um setor censitário inexistente, o sistema apresentará a mensagem: "Setor Censitário inexistente"

  • ZEIS: Informe o código da ZEIS, ou selecione no "combo-box" ao lado do campo.

  • Rota(*): Informe o código de uma rota existente e tecle "Enter", ou clique no botão "Pesquisar", que fica ao lado do campo. Neste caso será apresentada uma tela de "popup", onde será possível efetuar a pesquisa de rota no cadastro. Após a informação do código de uma rota existente, ou da seleção de uma rota na tela de pesquisa, o sistema apresentará o nome da rota no campo correspondente.

    • No caso de se informar uma rota inexistente, o sistema apresentará a mensagem: "Rota inexistente"

    • A rota deve pertencer ao setor comercial informado para a quadra.

Observação: Os campos obrigatórios estão marcados um um asterisco vermelho (*)

Funcionalidade dos botões:

Botão

Descrição

Utilize este botão para ativar as funcionalidades de pesquisa de uma localidade, um setor comercial, um distrito operacional, um setor censitário, ou uma rota. Deve ser utilizado quando não se conhece o código da informação que deseja inserir. Fica localizado ao lado do campo correspondente.

Ao clicar no botão, o sistema apresentará uma outra tela, onde será realizada a pesquisa.

Utilize este botão para limpar as informações existentes num determinado campo da tela. Eventualmente, poderá efetuar a limpeza de campos relacionados, com o objetivo de eliminar inconsistências. Por exemplo, nesta tela, ao limpar o campo "Localidade", o sistema limpará, automaticamente, os campos de "Setor Comercial" e "Quadra".
Utilize este botão para pesquisar as quadras existentes na base de dados, quando você não souber que numeração gerar para a quadra. Ao clicar no botão o sistema irá apresentar a relação das quadras já existentes, classificadas pelo código. Caso você clique no link existente no código da quadra, o sistema irá transerir o controle para a funcionalidade "Atualizar Quadra".
Utilize este botão para solicitar ao sistema a inserção da quadra na base de dados. Neste caso, é necessário que todos os campos estejam preenchidos corretamente. Caso exista alguma inconsistência, o sistema emitirá a mensagem de crítica correspondente. 
Utilize este botão para fazer com que a tela volte ao seu estado inicial de exibição. 
Utilize este botão para fazer com que o sistema encerre, sem salvar, o que está sendo feito, e volte para a tela principal.

Tela de Sucesso:

A tela de sucesso será apresentada após clicar no botão "Inserir", e não houver nenhuma inconsistência no conteúdo dos campos da tela.
O sistema apresentará a  mensagem abaixo, quando a inclusão da quadra tiver sido realizada com sucesso. 

"Quadra de número <<código da quadra>>, do setor comercial <<nome do setor comercial>>, da localidade <<nome da localidade>> inserida com sucesso."

O sistema apresentará tres opções após a inseção da quadra. Escolha a opção desejada clicando em algum dos "links" existentes na tela de sucesso:

  • Menu Principal  --> Para voltar à tela prinicipal.
  • Inserir outra quadra --> Para inserir uma outra quadra
  • Atualizar quadra inserida --> Para efetuar a atualização da quadra, recentemente, inserida

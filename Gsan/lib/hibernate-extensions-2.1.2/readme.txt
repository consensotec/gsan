Hibernate Extensions
====================
version 2.1.2, 25. April 2004

Instructions
------------

Unzip to installation directory.

tools/         contains ready-to-use hibernate-tools
               (hbm2java v2.1, class2hbm v2.0, ddl2hbm v2.0), 

hibern8ide/    the Hibernate IDE for browsing and querying
               objects with HQL, version 1.0

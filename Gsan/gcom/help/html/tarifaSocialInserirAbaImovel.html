Inserir Tarifa Social - Aba Imóvel

Esta opção do sistema permite a inserção da tarifa social para um imóvel.

O processo de inserção da tarifa social para um imóvel foi dividido em duas abas: 

  • Imóvel: Para identificação do imóvel a ser beneficado com a tarifa social
  • Tarifa Social: Para preenchimento das informações relacionadas com a tarifa social
As telas do tipo aba têm um comportamento padrão. Clique aqui para obter uma explicação mais detalhada do funcionamento das telas do tipo aba.
Agora nós vamos ver a ajuda para a aba "Imóvel".
Para ter acesso à ajuda das demais abas, clique no "link" correspondente, na sessão "Tópicos Relacionados".

Preenchimento dos campos

  • Matrícula(*): Informe o imóvel, para o qual, deseja efetuar o cadastramento da tarifa social. Para informar um imóvel você tem duas opções: 

    • a) Digitar a matrícula e teclar "Enter";

    • b) Pesquisar um imóvel --> Para ativar a opção de pesquisar imóvel clique no botão “Pesquisar”, que fica ao lado do campo. 

Uma vez selecionado o imóvel, o sistema apresentará as seguintes informações na tela:  

  • Endereço do Imóvel
  • Quantidade de economias existentes no imóvel
Então, confirmando as informações do imóvel, você deverá seguir para a próxima etapa do processo, para informar os dados da tarifa social propriamente dita. Você tem duas opções para avançar para a próxima etapa do processo:
  • Clicando no botão
  • Clicando sobre a aba "Tarifa Social"
Antes de avançar para a próxima etapa do processo de cadastratramento da tarifa social para o imóvel, o sistema irá efetuar um conjunto de validações para verificar se, para o imóvel selecionado, pode-se aplicar a tarifa social. Segue, abaixo, para exemplicar, um sub-conjunto das validações a serem realizadas:
  • Verificar se o imóvel selecionado ainda não está na tarifa social
  • Verificar a existência de Registro de Atendimento (RA) solicitando a inclusão da tarifa social para o imóvel
  • Verificar se o cliente proprietário do imóvel está cadastrado e se constam as informações do CPF e RG do cliente
  • Verificar se constam as informações do CPF e RG do cliente usuário; ou se ele já é beneficiário da tarifa social em outro imóvel
  • Verificar se todas as economias existentes no imóvel são residenciais
  • Verificar se o imóvel está com ligação de água ativada
  • Verificar se existe alguma anormalidade associada ao imóvel
  • Verificar se existem débitos para o imóvel
Observação: Os campos obrigatórios estão marcados um um asterisco vermelho (*)

Funcionalidade dos botões:

As telas do tipo aba apresentam dois tipos de botões, que estão separados por uma linha horizontal no final da tela.
Os botões que estão acima da linha horizontal, estão relacionados com a navegação entre as abas, e, os botões que estão abaixo da linha horizontal, têm efeito sobre todas as abas.

Botão

Descrição

Utilize este botão para ativar a funcionalidade "Pesquisar Imóvel". Deve ser utilizado quando você não souber qual é a matrícula do imóvel, para o qual, deseja inserir a tarifa social. 

Ao clicar no botão, o sistema apresentará uma outra tela, onde será possível realizar a pesquisa.
Estes botões estão na parte superior da linha horizontal, e devem ser utilizados para que o sistema avance para a próxima aba. Antes de passar para a próxima aba o sistema irá verificar se existe alguma inconsistência na aba atual. No caso de haver alguma inconsistência, o sistema apresentará a mensagem de crítica correspondente e não avançará para a próxima aba.

Utilize este botão para solicitar ao sistema a inserção da tarifa social para o imóvel. Neste caso, é necessário que todos os campos, de todas as abas, estejam preenchidos corretamente. Caso exista alguma inconsistência, o sistema emitirá a mensagem de crítica correspondente. 
Utilize este botão para fazer com que a tela volte ao seu estado inicial de exibição. 
Utilize este botão para fazer com que o sistema encerre, sem salvar o que está sendo feito, e volte para a tela principal.

Tela de Sucesso:

A tela de sucesso será apresentada após clicar no botão , e não houver nenhuma inconsistência no conteúdo dos campos de todas as abas do processo "Inserir Tarifa Social".
O sistema apresentará a  mensagem abaixo, quando a inclusão da tarifa social tiver sido realizada com sucesso. 

"Cliente de código <<código do cliente>> inserido com sucesso."

O sistema apresentará tres opções após a inserção do cliente. Escolha a opção desejada clicando em algum dos "links" existentes na tela de sucesso:

  • Menu Principal  --> Para voltar à tela principal do sistema.
  • Inserir outro cliente --> Para inserir um outro cliente.
  • Atualizar cliente inserido --> Para efetuar a atualização do cliente, recentemente, inserido.

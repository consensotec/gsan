Inserir Localidade

Esta opção do sistema permite a inserção de uma localidade na base de dados.

Para ter acesso a esta função você deve entrar na opção de menu: Cadastro --> Localização --> Localidade --> Inserir Localidade

Portanto, esta opção do sistema faz parte do conjunto de funcionalidades responsáveis pela estrutura de localização de imóveis.

  • Localidade
  • Setor Comercial
  • quadra

Preenchimento dos campos

  • Código(*): Informe o código da localidade que deseja inserir. Você pode, também, verificar se a localidade que você está pretendendo cadastrar já existe na base de dados. Clique no botão "Pesquisar"  que fica ao lado do campo, que será apresentada a tela "Pesquisar Localidade". Caso você encontre a localidade que está pretendendo cadastrar, e deseje atualizá-lo,  clique sobre o link existente no nome da localidade, que o sistema irá transferir o controle  para a opção "Atualizar Localidade".

  • Nome(*): Informe o nome da Localidade.

  • Endereço Localidade:
    • Para inserir o endereço da localidade clique no botão , que será aberta a tela "Informar Endereço", para você informar os dados referentes ao endereço que deseja adicionar (CEP, Logradouro, Bairro, ...). Após a formatação do endereço, clique no botão "Inserir" para voltar à tela  "Inserir Localidade" com a descrição do endereço formatada na tabela, e o botão   inabilitado. Clique aqui para obter ajuda para a tela "Informar Endereço".
    • Para atualizar as informações do endereço já inserido, clique sobre o link existente na descrição do endereço, que será aberta a tela "Atualizar Endereço", para você atualizar os dados referentes ao endereço cadastrado (CEP, Logradouro, Bairro, ...). Após a atualização do endereço, clique no botão "Atualizar" para voltar à tela "Inserir Localidade" com as informações atualizadas. Caso deseje fechar a tela "Atualizar Endereço", sem efetuar nenhuma alteração, clique no botão "Fechar". Clique aqui para obter ajuda para a tela "Atualizar Endereço".
    • Para remover o endereço, clique no botão "Remover" , que fica ao lado do endereço, na primeira coluna da tabela. Após clicar no botão o sistema irá abrir uma caixa de diálogo, solicitando a confirmação da remoção.  Clique no botão "Cancelar" para desistir da remoção, ou no botão "OK" para confirmar a remoção do endereço. Ao desistir da remoção, o sistema fechará a caixa de diálogo e voltará para a tela "Inserir Localidade" sem efetuar nenhuma modificação na tabela. Mas, ao confirmar a remoção, o sistema fechará a caixa de diálogo e voltará para a tela "Inserir Localidade" efetuando a remoção do endereço na tabela, e habilitando o botão  . 

  • Telefone: Informe, se existir, o número do telefone da Localidade, apenas com números positivos, e com 7 dígitos.

  • Ramal: Informe, se existir, o número do ramal do telefone da Localidade. Caso informe o ramal, o campo de telefone passa a ser obrigatório. 

  • Fax: Informe o número do Fax da Localidade, apenas com números positivos, e com 7 dígitos.

  • E-mail: Informe o endereço de e-mail da Localidade. deve respeitar a regra para formação de e-mail. 

  • Menor Consulo: Informe o valor do consumo mínimo para as economias existentes na Localidade.

  • Gerência Regional(*): Selecione a Gerência Regional, à qual, a Localidada está vinclulada. Ao selecionar a Gerência Regional o sistema irá popular o campo Elo com os elos vinculados à Gerência Regional. 

  • Elo: Este campo será populado após a seleção da Gerência Regional. Portanto, após selecionar a Gerência Regional, selecione o Elo, ao qual, a Localidada está vinclulada.

  • Classe(*): Selecione a Classe da Localidada. 
  • Porte(*): Selecione o Porte da Localidada. 

Observação: Os campos obrigatórios estão marcados um um asterisco vermelho (*)

Funcionalidade dos botões:

Botão

Descrição
Utilize este botão para adicionar o endereço da Localidade na tabela "Endereço Localidade". Ao ser acionado, será apresentada a tela "Informar Endereço", onde você deverá informar os dados do endereço da localidade 
Clique neste botão para efetuar a remoção do "Endereço da Localidade". Ao ser acionado, o sistema irá abrir uma caixa de diálogo, solicitando a confirmação da remoção.
Utilize este botão para fazer com que a tela volte ao seu estado inicial de exibição. Ao clicar neste botão o sistema limpará todos os campos informados até o momento.
Utilize este botão para fazer com que o sistema encerre, sem salvar o que está sendo feito, e volte para a tela principal.

Utilize este botão para solicitar ao sistema a inserção da localidade na base de dados. Neste caso, é necessário que todos os campos estejam preenchidos corretamente. Caso exista alguma inconsistência, o sistema emitirá a mensagem de crítica correspondente. 

Tela de Sucesso:

A tela de sucesso será apresentada após clicar no botão "Inserir", e não houver nenhuma inconsistência no conteúdo dos campos da tela.
O sistema apresentará a  mensagem abaixo, quando a inclusão da localidade tiver sido realizada com sucesso. 

"Localidade de código <<código da localidade>> inserida com sucesso."

O sistema apresentará tres opções após a inserção da localidade. Escolha a opção desejada clicando em algum dos "links" existentes na tela de sucesso:

  • Menu Principal  --> Para voltar à tela principal do sistema.
  • Inserir outra Localidade --> Para abrir a tela de inserção para outra localidade.
  • Atualizar Localidade Inserida --> Para efetuar a atualização das informações da localidade, recentemente, inserida.

Atualizar Quadra

Esta tela faz parte do processo "Manter Quadra", e permite que você atualize as informações da quadra.
Os campos de "Localidade", "Setor Comercial" e "Quadra" não podem ser alterados, porque compõem a chave de identificação da quadra.
Caso esta identificação esteja errada, pode ser por dois motivos:
  1. Você selecionou a quadra errada para alteração, neste caso deverá voltar para a tela anterior, ou para a tela de filtro, para selecionar a quadra correta;
  2. A idendificação, de fato, não deve existir. Neste caso você deve excluir esta quadra e incluir outra com a codificação correta. 
O sistema apresenta os campos da quadra preenchidos com as informações existentes no banco de dados.
Efetue as alterações que desejar, e clique no botão para solicitar ao sistema a atualização das informações na base de dados.

Prenchimento dos campos

  • Perfil da Quadra(*): Selecione um perfil no “combo-box” correspondente.

  • Rede de Água(*): Informe se não existe, existe, ou se a quadra é parcialmente coberta pela rede de água.

  • Rede de Esgoto(*): Informe se não existe, existe, ou se a quadra é parcialmente coberta pela rede de esgoto.

  • Os campos relacionados abaixo só deverão ser informados, caso exista rede de esgoto na quadra:

    • Sistema de Esgoto: Informe o código do sistema de esgoto, ou selecione no “combo-box” ao lado do campo.

    • Bacia: Informe o código do sistema de esgoto, ou selecione no “combo-box” ao lado do campo.

  • Distrito Operacional: Só deve ser informado, caso exista rede de água na quadra. Neste caso, informe o código de um distrito operacional existente e tecle “Enter”, ou clique no botão “Pesquisar”, que fica ao lado do campo. Neste caso será apresentada uma tela de “popup”, onde será possível efetuar a pesquisa dos distritos operacionais existentes no cadastro. Após a informação do código de um distrito operacional, ou da seleção de um distrito operacional na tela de pesquisa, o sistema apresentará o nome do distrito operacional no campo correspondente.

    • No caso de se informar um distrito operacional inexistente, o sistema apresentará a mensagem: Distrito Operacional Inexistente”

  • Setor Censitário: Informe o código de um setor censitário existente e tecle “Enter”, ou clique no botão “Pesquisar”, que fica ao lado do campo. Neste caso será apresentada uma tela de “popup”, onde será possível efetuar a pesquisa de setores censitários no cadastro. Após a informação do código de um setor censitário existente, ou da seleção de um setor censitário na tela de pesquisa, o sistema apresentará o nome do setor censitário no campo correspondente.

    • No caso de se informar um setor censitário inexistente, o sistema apresentará a mensagem: "Setor Censitário inexistente”

  • ZEIS: Informe o código da ZEIS, ou selecione no “combo-box” ao lado do campo.

  • Rota(*): Informe o código de uma rota existente e tecle “Enter”, ou clique no botão “Pesquisar”, que fica ao lado do campo. Neste caso será apresentada uma tela de “popup”, onde será possível efetuar a pesquisa de rota no cadastro. Após a informação do código de uma rota existente, ou da seleção de uma rota na tela de pesquisa, o sistema apresentará o nome da rota no campo correspondente.

    • No caso de se informar uma rota inexistente, o sistema apresentará a mensagem: "Rota inexistente”

    • A rota deve pertencer ao setor comercial informado para a quadra.

  • Indicador de Uso: Utilize este campo para modificar a situação da quadra de "Ativa" para "Inativa", e vice-versa.

Observação: Os campos obrigatórios estão marcados um um asterisco vermelho (*)

Funcionalidade dos botões

Botão

Descrição

Utilize este botão para ativar as funcionalidades de pesquisa de um distrito operacional, um setor censitário, ou uma rota. Deve ser utilizado quando não se conhece o código da informação que deseja inserir. Fica localizado ao lado do campo correspondente.

Ao clicar no botão, o sistema apresentará uma outra tela, onde será realizada a pesquisa.

Utilize este botão para limpar as informações existentes num determinado campo da tela. 


Utilize este botão para voltar para a tela anterior.
Utilize este botão para fazer com que a tela volte ao seu estado inicial de exibição. 
Utilize este botão para fazer com que o sistema encerre, sem salvar, o que está sendo feito, e volte para a tela principal.
Utilize este botão para solicitar ao sistema a atualização da quadra na base de dados. Neste caso, é necessário que todos os campos estejam preenchidos corretamente. Caso exista alguma inconsistência, o sistema emitirá a mensagem de crítica correspondente. 

Tela de Sucesso:

A tela de sucesso será apresentada após clicar no botão "Atualizar", e não houver nenhuma inconsistência no conteúdo dos campos da tela.
O sistema apresentará a  mensagem abaixo, quando a atualização da quadra tiver sido realizada com sucesso. 

“Quadra de número <<código da quadra>>, do setor comercial <<nome do setor comercial>>, da localidade <<nome da localidade>> atualizada com sucesso.”

O sistema apresentará duas opções após a atualização da quadra. Escolha a opção desejada clicando em algum dos “links” existentes na tela de sucesso:

  • Menu Principal  --> Para voltar à tela prinicipal.
  • Realizar outra manutenção de quadra --> Para efetuar a atualização de outra quadra
